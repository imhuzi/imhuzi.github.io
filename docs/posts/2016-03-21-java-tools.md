### Visual VM

Visual VM 是一款用来分析 Java 应用的图形工具，能够对 Java 应用程序做性能分析和调优。如果你使用的 java 7 或者 java 8，那么可以直接在 JDK 的 bin 目录找到该工具，名称为 [jvisualvm](http://visualvm.java.net/)。当然也可以在官网上自行下载
